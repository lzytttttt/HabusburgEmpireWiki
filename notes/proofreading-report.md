# 基础校对报告

## 校对日期

2026-04-29

## 校对范围

全部 9 个章节

## 校对原则

1. 只修正明显错别字
2. 只修正明显标点错误
3. 修正全角半角混乱
4. 删除多余空格
5. 修正明显重复字
6. 修正明显转换错误
7. 只做小幅修改不通顺的病句
8. 统一人名、地名、王号格式的明显不一致

## 主要修正类型

### 1. "的"与"地"混用

大量副词后面的"的"修正为"地"，例如：
- "光明正大的说出" → "光明正大地说出"
- "不停的骚扰" → "不停地骚扰"
- "成功的击败" → "成功地击败"
- "习惯性的将" → "习惯性地将"

### 2. 地名不一致

- "波希米亚" → "波西米亚"（全文统一）
- "特兰西瓦西亚" → "特兰西瓦尼亚"

### 3. 错别字

- "重现确立" → "重新确立"
- "壁垒深严" → "壁垒森严"
- "苟延馋喘" → "苟延残喘"
- "编篡" → "编纂"
- "组名的" → "著名的"
- "夺去" → "夺取"
- "排特使" → "派特使"

### 4. 多余/缺失字

- "的的合法宣称权" → "的合法宣称权"
- "古老的的日耳曼" → "古老的日耳曼"
- "军队军队" → "军队"
- "获得了了" → "获得了"
- "奥地利最强大的几个封建家族奥地利最强大的几个封建家族" → "奥地利最强大的几个封建家族"

### 5. 全角/半角修正

- 半角括号 → 全角括号（人名、年份等）
- 多余空格删除

### 6. 语法修正

- "克罗地亚并在" → "克罗地亚在"
- "成立维持" → "已成为维持"
- "在1438年了" → "在1438年"
- "阿尔弗雷德" → "阿尔布雷希特"
- "1514年" → "1541年"（日期错误）
- "1444年" → "1644年"（日期错误）

### 7. 其他修正

- "Safari Persia" → "萨法维波斯"
- "编篡法典的法学家伊什特万Verboczy" → "编纂法典的法学家伊什特万·韦伯奇（Verboczy）"

## 未修改内容
