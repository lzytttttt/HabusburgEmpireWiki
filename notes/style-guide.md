# 术语风格指南

> 本文件记录术语翻译的统一规范。在校对过程中逐步补充。

## 人名翻译规则

- 哈布斯堡家族成员：费迪南德、马克西米连、鲁道夫、查理等
- 匈牙利人名：马蒂亚斯、西吉斯蒙德、査珀莱等
- 波西米亚人名：胡斯、乔治等
- 奥斯曼人名：苏莱曼、穆罕默德等

## 地名翻译规则

- Bohemia → 波西米亚（不使用"波希米亚"）
- Transylvania → 特兰西瓦尼亚
- Hungary → 匈牙利
- Austria → 奥地利
- Croatia → 克罗地亚
- Moravia → 摩拉维亚
- Silesia → 西里西亚
- Lusatia → 卢萨蒂亚
- Tyrol → 蒂罗尔
- Vienna → 维也纳
- Prague → 布拉格
- Budapest → 布达佩斯（历史语境中分别使用"布达"和"佩斯"）

## 标点规范

- 人名、年份等使用全角括号：（）
- 外文原文使用半角括号：()
- 书名号使用：《》
- 引号使用：""

## 格式规范

- 副词后面使用"地"而非"的"
- 避免重复字
- 全角/半角统一
