# 哈布斯堡帝国的崛起 译稿整理项目

## 项目简介

本项目是对玛玛吉（Mamatey, Victor S.）所著《Rise of the Habsburg Empire, 1526-1851》中译初稿的整理与 wiki 化工作。

## 目标

- 将初稿整理为 Markdown wiki
- 校对、统一术语
- 建立人物索引、地名索引、时间线
- 整理待考问题

## 文件结构

```
manuscript/original/    ← 原始文稿，禁止修改
manuscript/working/     ← 工作副本，用于校对整理
wiki/                   ← Markdown wiki 输出
wiki/chapters/          ← 按章节拆分的 wiki 页面
notes/                  ← 工作笔记、校对规则、待考问题
data/                   ← 结构化数据（术语表、人名表等）
scripts/                ← 自动化脚本
```

## 工作原则

1. 原始文稿永不修改
2. 校对在工作副本中进行
3. 每次只处理一个章节
4. 不确定之处记录到 `notes/unresolved.md`
